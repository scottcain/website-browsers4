  • December 09, 2007: New interface for the Textpresso literature search
    Textpresso -- the software that powers the WormBase Literature Search -- has a new user interface. Available directly from the WormBase home page (select the "Literature" option), Textpresso searches individual sentences in the full text of 9461 papers and 24,526 abstracts.
